# Pedal Maestro - Taller de Bicicletas

## Descripción
Landing page para un taller mecánico de bicicletas que ofrece servicios de reparación, mantenimiento preventivo y venta de repuestos.

## Problemática abordada
Un emprendimiento local (taller de bicicletas) necesita presencia en línea para promocionar sus servicios y captar clientes. La página permite visualizar los servicios, bicicletas destacadas y contactar mediante formulario.

## Tecnologías utilizadas
- HTML5 semántico
- CSS3 (con variables, flex/grid, diseño responsivo)
- Bootstrap 5 (framework CSS)
- Git para control de versiones
- GitHub para alojamiento remoto

## Uso de IA (GitHub Copilot)
Se utilizó GitHub Copilot durante el desarrollo para:
- Generar la estructura de las tarjetas de servicios con Bootstrap.
- Crear el formulario de contacto accesible con etiquetas asociadas.
- Sugerir reglas CSS para foco visible y paleta de colores con contraste AA.

Los comentarios en el código indican las secciones asistidas por IA.

## Cómo visualizar
Clona el repositorio y abre `index.html` en tu navegador.
